Dernière nouvelle 2015-03-16

Nouvelles iso en 32 et 64 bits disponible.

J'ai le grand plaisir de vous proposer des nouvelles iso 32 bits et 64 bits:

en 64 bits et en 32 bits

L'installateur a encore été simplifié. Désormais si vous configurez la partition de destination et que vous la formatez avant de lancer l'installateur, aucune question ne sera posée et l'installation débutera immédiatement cad l'affaire de 15 secondes.

Le service Openssh est désormais disponible dès l'install et donc permet d'accéder à votre NuTyX fraichement installée depuis un terminal de la machine host (si test en virtualbox avec carte réseau configuré en pont). Très utile si vous avez deux PC.

Le wifi devrait fonctionné après installation de xfce4 / kde.

Nouveaux paquets:

    • Nouvelles commandes:

    cards files
    affiche la liste de(s) fichier(s) du paquet. 
    cards query
    affiche tous les paquets et les fichiers contenant le pattern ( equivalent à pkginfo -o ) 
    cards install
    Permet d'installer un fichier si présent sur la machine. Le fichier doit être un paquet téléchargé. Le nom n'est pas important, cards vérifie son contenu avant l'install. 
    cards install -f <paquet> | <file>
    Permet d'installer un paquet si doublon. (équivalent à pkgadd -f ).

    Les commandes:

    cards sync -i
cards list -i
cards info -i 
cards diff -b
    sont simplifiées et deviennent: 
    cards sync
cards list
cards info 
cards diff
