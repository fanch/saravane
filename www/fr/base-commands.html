Les commandes de base de NuTyX

Afin de profiter pleinement de votre NuTyX fraîchement installée, voici les commandes indispensables que vous allez utiliser quotidiennement. Cet article reprend les commandes essentielles et spécifiques à NuTyX. Il est conseillé de les bien les connaître.

NuTyX ne dispose pas (encore) d'outils graphiques pour son administration. Vous allez donc utiliser le terminal comme interface. Si vous n'y êtes pas habitué, ce n'est pas grave. Cet article vous expliquera pas à pas les commandes à taper au clavier.

Sous NuTyX (contrairement à d'autres systèmes d'exploitation) la casse doit être respectée, il faut donc toujours veiller à taper les commandes en minuscules.

Commençons donc par ouvrir un terminal... ça y est ? alors on peut commencer

Une commande peut être composée d'un seul mot dans le cas où aucune option n'est nécessaire. Mais la majorité des commandes peut être entrée à l'aide d'options diverses. Sachez par exemple que l'option la plus utile sera (en fonction de la commande)

nomdelacommande -h

le -h permet d'obtenir un aperçu des possibilitées de la nomdelacommande, --help ou help sont aussi des options permettant l'affichage de l'aide.

Il faut savoir que certaines commandes mentionnées dans cet article sont en réalité des alias, c'est à dire un racourci de la commande originale. Si vous souhaitez connaître la liste des alias, utilisez la commande: 
alias
Bien que ce ne soit absolument pas necessaire, si vous souhaitez malgré tout modifier les alias définis dans votre NuTyX, vous devez éditer le fichier .bashrc qui se trouve dans le dossier personnel du compte concerné. Encore une fois, cette opération n'est pas du tout indispensable et il est vivement conseillé de bien connaître le principe avant de modifier les alias prédéfinis
/root/.bashrc
En voici un extrait: 
if [ -f "/etc/bashrc" ] ; then
  source /etc/bashrc
fi
if [ $EUID -eq 0 ]; then
         alias del='cards remove'
         alias get='cards install'
         alias up='cards install -u'
         alias check='cards sync -i;cards diff -b'
         alias search='cards search'
else
         alias del='sudo cards remove'
         alias get='sudo cards install'
         alias up='sudo cards install -u'
         alias check='sudo cards sync -i;sudo cards diff -b'
         alias search='sudo cards search'
fi
alias ll='ls -l'
alias l='ls -alh'
alias duh='du -h --max-depth=1'
alias dfh='df -h'
alias pkgfind='cards list -i | grep -i'
# End ~/.bashrc

Recherche d'un paquet

Pour rechercher un paquet sous NuTyX, c'est très simple, la commande :

search chaîne_à_trouver

retournera tous les paquets dont le nom ou la description contient la chaîne_à_trouver.Par exemple si vous souhaitez connaître les navigateurs disponible (en binaire)

search browser

devrait retourner quelque chose comme :

chromium           The open-source project behind Google Chrome, an attempt at creating a safer, faster, and more stable browser. Include PepperFlash and libpdf plugins
firefox            Standalone web browser from mozilla.org
...

Ajout d'un utilisateur

Pour ajouter un utilisateur sous NuTyX, c'est également très simple, la commande :

nu

permet d'ajouter un utilisateur, elle peut être utilisée sous 2 formes avec et sans options, si elle est utilisée sans options, 2 questions vous seront demandées:

Nom de l'utilisateur: francois
Description de l'utilisateur: François Perrin

Vous devez bien noter que le nom d'utilisateur ne peut pas contenir certains caractères et ne peut pas contenir d'espace ni de majuscule.

Si par contre vous souhaitez utiliser la commande avec ses options, ces derniers sont dans l'ordre la description de l'utilisateur suivi du nom de l'utilisateur:

nu "François Perrin" francois

Ajout d'un logiciel

Pour ajouter un logiciel, un alias existe pour vous, la commande get suivi du ou des noms des logiciels à installer sera utilisée :

get firefox

Le téléchargement de l'application firefox commence imnmédiatement, une fois le paquet téléchargé, ses dépendances seront à leur tour téléchargées. Quand le téléchargement est terminé, tous les paquets seront installés dans le bon ordre.

Vérifier si il y a des mises à jour disponible

La commande check permet d'obtenir le dernier état des mises à jour disponibles sur votre système.

check
Synchronizing /var/lib/pkg/saravane/desktop from http://..
/var/lib/pkg/saravane/desktop//MD5SUM
..
Differences between installed packages and the depot of binaries:

Package                        Installed                    Available in the depot of binaries

sylpheed                      3.0.1-1                     3.2.0-1

Dans cet exemple sylpheed peut être mis à jour.

Mettre à jour un logiciel

Pour mettre à jour un logiciel, la commande cards install -u suivi du logiciel sera utilisée. Il faut evidement que la mise à jour soit possible :

cards install -u sylpheed

Si vous avez l'alias up configuré, il fera la même chose.

Conclusion

Vous l'avez compris les seules commandes à connaître pour commencer à utiliser NuTyX sont donc les 2 commandes :

nu
cards

Cette dernière est d'ailleurs expliquée plus en détail via son aide:

cards

Je vous souhaite beaucoup de plaisir avec votre nouvelle NuTyX installée.
