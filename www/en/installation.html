Installation

Make shure you have enough space on your hard disk. A size of 8Go to 15Go will make it.

Download the last version of NuTyX

The actual version of NuTyX is named saravane 15.05. The available iso can be used to install all the packages from internel. It's also to used it for a minimum of maintenance on your machine before the actual installation of NuTyX.

The size of the isos are +- 202 Mb for the 64 bits and +- 197 Mb for the 32 bits.

All the available iso can be download at: http://downloads.nutyx.org

Create a bootable USB key

If you are currently running GNU/linux on your machine:

In root

With the help of fdisk, find out where your usb KEY is located:

fdisk -l

Then by using the command dd, create a bootable key:

dd if=NuTyX_x86_64-saravane-20150502.iso of=/dev/sdX

The sdX value is given by the command:

fdisk -l

If you are under Windows, the easiest way is to burn the ISO on a CD.

Typical installation of NuTyX on a single partition

This topic will explain you how to install from scratch the NuTyX distribution on your PC

The default password is nutyx. As soon you successfully been login first time, it will be required to change the password before you can go to the prompt.

Installation of the boot process

This topic will explain you how to install from scratch the configuration of the boot process of your PC

If you used the latest grub version 2.XX , the following entry can be added to your grub.cfg:

menuentry "NuTyX sur /dev/sda1" {
    set root=(hd0,1)
    linux /boot/kernel root=/dev/sda1 ro quiet
    initrd /boot/initrd
    }

    menuentry "Windows 8" {
    set root=(hd0,3)
    chainloader (hd0,3)+1
    }
